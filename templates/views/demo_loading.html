Loading Indicator & Cancel

While AI is processing your request, a loading animation appears and you can cancel:

  • Animated dots show activity
  • Send button changes to Cancel (red X)
  • Click Cancel to stop the request
  • Replaces with actual content when ready
  • Error message shown if request fails

Auto-Clear & Disable Input

After sending a message, the input area is managed automatically:

  • Textarea content cleared
  • File attachments removed
  • Input height reset
  • Input disabled during AI response
  • Prevents multiple submissions
Try it: Type a message and click send. The input clears, and the send button becomes a red Cancel button. Try clicking it to cancel the response!
